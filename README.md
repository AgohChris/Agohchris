### 🛒 Totogourmand API

#### ⚙️ Core Features
- JWT Authentication (access + refresh + revoke)
- Order, cart, blog & user management
- Robust input validation

#### 🛡️ Security Engineering
- XSS protection (HTML sanitization with `bleach`)
- IDOR prevention (strict ownership checks)
- JWT revocation on sensitive actions
- Rate limiting on authentication endpoints
- Secure headers & hardened settings
- Race condition prevention (`select_for_update`)
